Welcome to WordPuz Pages.

A game of ability and speed to search of words in the window-box in short time.
Are provided several way of level, that see the time of game reduce in base to choice.It is also provided the multi language, that may make it in Italian or English.

Screenshots

     

Features

  • Several way of game (Level)
  • Manage Scores
  • Select several languages
  • Style as like RuzzleGame for Mobile

Development details

WordPuz has been developed by miclen in October 2013 for a exam project in Technique Advanced Programming(TPA) in University of Salerno. I have used as programming language Java, with Technique Thread, Swing and i18n (Locale).
